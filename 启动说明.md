# 如何启动AI领域多媒体推荐系统

## 快速启动

### 方式1：使用启动脚本

```bash
cd /Users/macbook/Desktop/AI-Pedia/Project
./start.sh
```

启动脚本会自动：
- 检查Python环境
- 检查并安装依赖
- 创建必要目录
- 启动应用

### 方式2：手动启动

```bash
# 1. 进入项目目录
cd /Users/macbook/Desktop/AI-Pedia/Project

# 2. 安装依赖（首次运行需要）
pip3 install -r requirements.txt

# 3. 启动应用
python3 app.py
```

## 启动后

应用启动成功后，你会看到类似这样的输出：


## 访问应用

在浏览器中打开：
- **http://localhost:5000**
- 或 **http://127.0.0.1:5000**

## 停止应用

在运行应用的终端窗口中，按：
```
Ctrl + C
```

## 常见问题

### 1. 端口被占用

如果5000端口被占用，可以修改 `app.py` 最后一行：

```python
app.run(debug=True, port=5001)  # 改为其他端口
```

### 2. 依赖未安装

如果提示模块找不到，运行：
```bash
pip3 install -r requirements.txt
```

### 3. 权限问题

如果启动脚本无法执行，运行：
```bash
chmod +x start.sh
```

## 使用流程

1. **启动应用** → 运行 `python3 app.py` 或 `./start.sh`
2. **打开浏览器** → 访问 http://localhost:5000
3. **上传文件** → 上传包含至少10个txt/pdf文档的AI/机器学习相关文档zip文件
4. **等待处理** → 系统自动提取AI相关关键词并搜索AI领域资源
5. **下载结果** → 下载推荐结果的zip文件（已过滤不相关内容和联系方式）

## 注意事项

- **文档主题**：建议上传AI/机器学习相关的文档（如机器学习、深度学习、神经网络等）
- **文件格式**：支持txt和pdf格式（PDF会自动转换为TXT）
- **文件数量**：上传的zip文件需包含至少10个txt/pdf文档
- **网络要求**：确保网络连接正常（需要访问外部资源）
- **处理时间**：处理时间取决于文档数量和网络速度（通常2-5分钟）
- **推荐内容**：系统会自动过滤非AI相关内容（如报告、政策文档等）
